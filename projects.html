Projects

Footy Visions is a personal app project built in Flutter using Dart and Google ML Kit. The app detects and tracks a red AFL football in real time, fits a parabolic trajectory, and displays a tracer line with a predicted kick distance — similar to goal-tracking visuals in sports broadcasts.

The AI Assistant is a voice-driven smart hub that seamlessly connects and controls everyday devices such as TVs, laptops, and gaming consoles. Designed with a natural-sounding voice and intelligent speech recognition, it performs real-time actions based on user commands — from launching apps to playing media or adjusting settings. Built with a modular backend architecture and intuitive voice interface, the project showcases advanced integration between AI, automation, and connected devices — demonstrating strong skills in software engineering, system design, and user experience innovation.
